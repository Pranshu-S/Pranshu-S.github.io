Project Overview

Quadcopter Failure Algorithm

Unlike a fixed-wing drone, failure of motors in a multirotor craft may cause safety problems. Numerous published articles have proposed solving this problem by redesigning the control law or control gain


Methods were studied to implement and develop a failsafe Algorithm for Quadcopters in case of single propeller failure. A solution for quadcopters losing three, two opposing, or one propeller in uses a linear quadratic regulator (LQR) to rotate the quadcopter about a specified axis that is periodically fixed in both inertial and body frame.


This project aims to stabilize and land the quadcopter autonomously after an unexpected motor failure which means that all computations must be performed in flight. The goal is to still achieve fault tolerant return and landing despite limitations in sensor accuracy and computing power


This project was performed under FARC NITK.

Sahayak Bot - Eyantra

This was done developed during Eyantra 2020-21 held by IITB.


The problem statement required to develop an Autonomous Ground Vehicle (AGV) and make it capable of autonomously traversing an indoor environment to assist moving objects from one place to another. Project Overview

Home Cleaning Robot Using Home Navigation

The article presents a home servicing robot with home navigation. The user can command the robot via a program developed using MIT app inventor and the Android Studio. The commands from the user are sent via Bluetooth to the Arduino Mega for processing. It is also equipped with a navigation system using a map protocol for moving the bot.


The proposed problem statement was to develop an Autonomous Bot and make it capable of autonomously traversing an indoor house to assist in cleaning rooms. The room locations were pre-defined and the user would just require to define the location of which room is to be cleaned using a Mobile App and the bot would do as instructed.


This project was completed for the National Knowledge Quest Competition 2018 held by BITS Pilani, Dubai.

Project Overview

Swarm Intelligence

The problems involved in swarm robotics research can be classified into two classes. One class of the problems is based on the patterns, such as accumulation, cartography, migration, self-organizing grids, deployment of distributed agents and area coverage. Another class of problems focuses on the entities in the environment, e.g. searching for the targets, detecting odour sources, finding the ore veins in wild field, foraging, rescuing the victims in disaster areas, etc.


This project aims to extend its deliverables to a broad timeline so as to provide a platform for studying and analysis of swarm behaviours in different environments and return feedbacks. By developing a base model for the autonomous control system of individual bots, we can study how presence of multi-robots can affect the performance and efficiency in completing the given task.


The aim is to explore an unknown environment using robotic swarms. This project shall use custom – built simple bots that are decentralized and homogenous to cover and explore a given territory to find targets and tasks of higher precedence as per requirement need. In doing so, the frequent problem with dynamic swarm task allocation will be overcome and the swarm will show strategy adaptation with respect to the situation at hand.


This project was performed under FARC NITK.
